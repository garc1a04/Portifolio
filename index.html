PROJECTS

Simom Game

Simon Game (ou Genius), um jogo de memorização de padrões de cores e sons. Foi meu primeiro projeto prático com manipulação do DOM (Document Object Model), explorando eventos, funções e interações visuais e sonoras.

Subject index

Projeto acadêmico da disciplina de Estrutura de Dados. O sistema permite inserir textos manualmente ou via arquivos `.txt` e cria automaticamente um índice com as palavras selecionadas. Foi desenvolvido utilizando Árvore AVL e Tabela Hash.

Rating Component

Projeto realizado através da plataforma Frontend Mentor. Um componente simples de avaliação que me ajudou a praticar e reforçar conceitos de HTML e CSS.

DogTin

Um site fictício de encontros para cachorros, criado para exercitar o uso do framework Bootstrap. Neste projeto trabalhei com responsividade, componentes prontos e layout moderno.

Github Finder

Aplicação para buscar perfis no GitHub, utilizando a API pública da plataforma. Desenvolvido com Node.js, Express.js e consumo de APIs, esse projeto reforçou meus conhecimentos em back-end e integração de serviços.
